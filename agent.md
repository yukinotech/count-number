## 项目概览

这是一个基于 Vite 构建的 React + TypeScript 前端项目。

## 技术栈

- **核心框架**: React 19.2.0
- **语言**: TypeScript ~5.9.3
- **构建工具**: Vite ^7.2.4
- **包管理器**: pnpm

## 开发命令

- **启动开发服务器**: `pnpm dev`
- **构建生产环境**: `pnpm build`
- **预览生产构建**: `pnpm preview`
- **代码检查**: `pnpm lint`

## 项目结构

- `src/`: 源代码目录 (App.tsx, main.tsx)
- `public/`: 静态资源目录
- `index.html`: 入口 HTML 文件
- `vite.config.ts`: Vite 配置文件
